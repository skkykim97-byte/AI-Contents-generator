# 롯데백화점 HTML 콘텐츠 생성기

템플릿 기반으로 HTML 콘텐츠를 자동 생성하는 Streamlit 앱입니다.

## 🚀 설치 및 실행

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 앱 실행
```bash
streamlit run app.py
```

브라우저에서 자동으로 `http://localhost:8501`이 열립니다.

## 📋 주요 기능

### 지원 템플릿
1. **이벤트/프로모션 배너** 🎉
   - 프로모션, 할인 이벤트용 배너
   - 이미지, 제목, CTA 버튼 포함

2. **랜딩 페이지 블록** 📄
   - 상품/서비스 소개 페이지
   - Hero 섹션 + 특징 카드 + CTA

3. **카드형 콘텐츠** 🎴
   - 1~6개 카드 생성 가능
   - 상품 목록, 카테고리 소개 등

4. **공지/안내형** 📢
   - 공지사항, 안내 메시지
   - 강조 문구 삽입 가능

### 핵심 기능
- ✏️ **직관적인 폼 입력**: 템플릿별 필수 항목만 입력
- 👁️ **실시간 미리보기**: 데스크톱/모바일 뷰 지원
- 📥 **즉시 다운로드**: HTML 파일로 저장
- 📋 **코드 복사**: CMS에 바로 붙여넣기
- 🎨 **커스터마이징**: 색상, 이미지, 텍스트 자유 변경

## 🎯 사용 시나리오

### 마케터/기획자
1. 템플릿 선택
2. 콘텐츠 입력 (제목, 이미지, 링크 등)
3. HTML 생성 버튼 클릭
4. 미리보기 확인 후 다운로드

### 퍼블리셔
- 생성된 HTML 검수 및 미세 조정
- 고난도/커스텀 작업에 집중 가능

## 🛠️ 기술 스택

- **Streamlit**: 웹 인터페이스
- **Python**: 백엔드 로직
- **HTML/CSS**: 템플릿 생성

## 📝 템플릿 확장

새로운 템플릿 추가 방법:

1. `TEMPLATES` 딕셔너리에 새 템플릿 정의
2. 입력 필드 추가 (if 분기)
3. HTML 생성 함수 작성 (`generate_xxx_html`)
4. 생성 버튼 로직에 분기 추가

## 🔧 커스터마이징

### 색상 변경
- 앱 헤더 그라데이션: `.main-header` CSS 수정
- 버튼 스타일: `.cta-button` CSS 수정

### 템플릿 스타일
각 `generate_xxx_html` 함수 내 `<style>` 태그 수정
